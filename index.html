    A better YouTube search, with the filters YouTube hides

    zerodig is a free, advanced YouTube search tool. It exposes every filter the YouTube Data API offers and adds the ones YouTube's own search will not: filter by view count (down to near zero-view videos), upload date range, duration, likes, definition, license and category, then sort results any way you like. Search videos, channels and playlists in one place.

    Prefer no setup? Switch to YouTube URL mode: it builds a youtube.com search link straight from YouTube's own filter parameters, so you get better YouTube search filters with no API key and no quota.

    Everything runs in your browser. No account, no server, no tracking. Read the docs for the full filter reference and setup guide.
